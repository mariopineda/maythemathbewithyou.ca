This website was handcrafted lovingly and mindfully using the most versatile, most low tech and most powerful Open Source tools of the trade. Why would one want to do this when you could use the latest version of [insert name of any propritary, expensive and bloated software]? Because...

  • simple is beautful, less likely to break, easier to fix when it does break and comes with no unwanted baggage.
  • Open Source is fair and free (free as in freedom and as in your favourite beverage).
  • Open Source is socially responsible and immensely powerful.
  • I am a computer nerd taking pleasure in crafting code in a terminal window.

These pages were created using the Twitter Bootstrap 3.0.0 framework based on a heavily modified United template (source available at GitHub) from Bootswatch. The mathematics is typset using MathJax, the site is generated using the blog-aware static site generator Jekyll and hosted freely and openly at GitHub. The source markup is under, well, git Source Control Management.
