Determine the number of x-intercepts, the type of turning point(s) and the end behaviour of the following polynomials.

Year Carbon dioxide emissions (Megatonnes)
1990 15.5
1995 21.5
2000 25.6
2005 35.5
2010 53.9
2015 70.8
2017 80.5

A.

Quadratic function. Number of x-intercepts: 2, Turning points: Single global maximum, End behaviour: $Q3 \rightarrow Q4$.

B.

Cubic function. Number of x-intercepts: 1, 2 or 3, Turning points: Single local maximum and single local minimum, End behaviour: $Q2 \rightarrow Q4$.

C.

quadratic function. Number of x-intercepts: 0, Turning points: Single global maximum, End behaviour: $Q3 \rightarrow Q4$.

D.

Quadratic function. Number of x-intercepts: 1, Turning points: Single global minimum, End behaviour: $Q2 \rightarrow Q1$.

D.

Linear function. Number of x-intercepts: 1, Turning points: none, End behaviour: $Q2 \rightarrow Q4$.
