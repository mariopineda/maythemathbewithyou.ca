Solve $\triangle ABC$ where $\angle BAC = 70^{\circ}$, $\angle ACB = 35^{\circ}$, $b=20.5$ cm and $c=12.5$ cm only using the Sine Law. Round angles to the nearest degree and sides to nearest tenth.

{% if true %}

One possible way of solving $\triangle ABC$.

Find $a$: $\displaystyle\frac{a}{\sin(70^{\circ})} = \displaystyle\frac{12.5}{\sin(35^{\circ})}$, $a=\sin(70^{\circ}) \times \displaystyle\frac{12.5}{\sin(35^{\circ})}$, $a=20.47\ldots \approx 20.5$ cm

Find $\angle B$: $\displaystyle\frac{\sin(B)}{20.5} = \displaystyle\frac{\sin(35^{\circ})}{12.5}$, $\sin(B) = 20.5 \times \displaystyle\frac{\sin(35^{\circ})}{12.5}$, $\angle B = 70.16\ldots \approx 70^{\circ}$
