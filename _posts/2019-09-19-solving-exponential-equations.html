Solve the following exponential equations.

  1. $5^{2x+3} = 5^7$
  2. $7^{x-2} = 353$
  3. $3^{5x-1} = 81^{3x}$
  4. $5^{4a+2} = 25^a$
  5. $2^{-1} = 64$
  6. $10^{10x-2} = 1000^{20x-42}$
  7. $8^{3x} = 4^{1-x}$
{% if true %}
  1. $5^{2x+3} = 5^7 \Rightarrow$ drop the bases: $2x+3=7$, solve for $x$: $x=2$
  2. $7^{x-2} = 353 \Rightarrow$ common bases: $7^{x-2} = 7^3$, drop the bases: $x-2=3$, solve for $x$: $x=5$
  3. $3^{5x-1} = 81^{3x} \Rightarrow$ common bases: $3^{5x-1} = (3^4)^{3x}$, Power of a Power Law:$3^{5x-1} = 3^{12x}$, drop the bases: $5x-1=12x$, solve for $x$: $x=-\displaystyle\frac{1}{7}$
  4. $5^{4a+2} = 25^a$
  5. $2^{-1} = 64$
  6. $10^{10x-2} = 1000^{20x-42}$
  7. $8^{3x} = 4^{1-x}$
{% endif %}
