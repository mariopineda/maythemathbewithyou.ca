Evaluate without a calculator.

  1. $$\frac{3}{12} + \frac{5}{12}$$
  2. $$\frac{19}{12} - \frac{4}{3}$$
  3. $$\frac{5}{8} \times \frac{3}{2}$$
  4. $$\frac{6}{5} \div \frac{2}{8}$$
  5. $$\frac{14}{4} - \frac{8}{6}$$
  6. $$\frac{1}{4} \div \left( \frac{5}{8} + \frac{2}{4} \right)$$
{% if true %}
  1. $$\frac{3}{12} + \frac{5}{12} = \frac{8}{12} = \frac{2}{3}$$
  2. $$\frac{19}{12} - \frac{4}{3} = \frac{19}{12} - \frac{16}{12} = \frac{3}{12} = \frac{1}{4}$$
  3. $$\frac{5}{8} \times \frac{3}{2} = \boxed{\frac{15}{16}}$$
  4. $$\frac{6}{5} \div \frac{2}{8} = \frac{6}{5} \times \frac{8}{2} = \frac{48}{10} = \frac{24}{5}$$
  5. $$\frac{14}{4} - \frac{8}{6} = \frac{42}{12} - \frac{16}{12} = \frac{26}{12} = \frac{13}{6}$$
  6. $$\frac{1}{4} \div \left( \frac{5}{8} + \frac{2}{4} \right) = \frac{1}{4} \div \left( \frac{5}{8}+\frac{4}{8} \right) = \frac{1}{4} \div \frac{9}{8} = \frac{1}{4} \times \frac{8}{9} = \frac{8}{36} = \frac{2}{9}$$
{% endif %}
